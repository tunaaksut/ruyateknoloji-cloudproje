# Rüya Teknoloji Web Sitesi - Cloud Deployment

Bu proje, Google Cloud Platform'da Ubuntu tabanlı bir VM üzerinde Apache ve PHP kullanarak canlı web sitesi yayını yapmayı kapsamaktadır.

## Teknolojiler
- Google Cloud Compute Engine (VM)
- Ubuntu 20.04 LTS
- Apache Web Server
- PHP 7.x

## Proje Özellikleri
- VM kurulumu ve yapılandırması
- Apache kurulumu ve site konfigürasyonu
- PHP destekli web sitesinin yayına alınması
- Dosya izinleri ve güvenlik ayarları
- 403 Forbidden hata çözümü

## Nasıl Çalıştırılır?
1. Google Cloud Platform'da Compute Engine üzerinde Ubuntu VM oluşturun.
2. Apache ve PHP kurulumunu yapın.
3. Web sitesi dosyalarını VM'ye yükleyin.
4. Apache konfigürasyon dosyasını güncelleyerek siteyi aktif hale getirin.
5. Dosya izinlerini kontrol edin.
6. Apache servisini yeniden başlatın.

## İletişim
Proje ile ilgili sorular için bana ulaşabilirsiniz.
